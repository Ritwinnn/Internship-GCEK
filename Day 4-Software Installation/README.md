# Day 4 — Software Setup

## Overview
On Day 4 of my internship, I focused on setting up all the necessary software tools required for upcoming ESP32 programming, PCB design, and embedded systems development. This setup is essential for hardware-based tasks in the internship.

## Objective
Set up and verify the software tools required for the internship.

## Software Installed

### 1. KiCad
- Open-source PCB design software
- Used for schematic capture and PCB layout
- Helps in designing electronic circuits and printed circuit boards

### 2. Eclipse Temurin JDK 25
- Java Development Kit
- Required for running Java-based applications and development tools
- Ensures compatibility with software dependencies

### 3. CP210x USB Driver
- USB-to-UART communication driver
- Required for connecting ESP32 boards to the computer
- Allows the system to detect and communicate with ESP32 hardware

### 4. VS Code
- Lightweight and powerful code editor
- Used for writing, editing, and managing source code
- Supports extensions for embedded development

### 5. PlatformIO
- Embedded systems development platform
- Installed as a VS Code extension
- Used for programming, compiling, and uploading code to ESP32 boards

## Steps Performed
1. Downloaded and installed all required software.
2. Verified successful installation of each tool.
3. Configured VS Code with PlatformIO extension.
4. Installed CP210x drivers for ESP32 connectivity.
5. Tested the setup for future hardware programming.

## Key Learnings
- Importance of development environment setup
- Role of PCB design tools in electronics
- ESP32 communication setup
- Embedded programming workflow using PlatformIO

## Outcome
Successfully installed and verified all required software and drivers for future ESP32 and hardware-based sessions.
