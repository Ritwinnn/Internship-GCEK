# Day 2 — Firebase Hosting Deployment

## Overview
On Day 2 of my internship, I learned how to deploy my personal portfolio website using Firebase Hosting. This gave me practical experience in cloud deployment and hosting static websites.

## Objective
- Understand Firebase Hosting
- Set up Firebase CLI
- Deploy a static website
- Make the portfolio accessible online

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Firebase Hosting
- Git & GitHub

## Steps Completed

### 1. Installed Firebase CLI
```bash
npm install -g firebase-tools
2. Logged into Firebase
firebase login
3. Initialized Firebase Hosting
firebase init hosting

Configuration included:

Selecting the Firebase project
Setting the public directory
Configuring as a single-page/static app
4. Added Portfolio Files

Project files:

index.html
style.css
script.js
5. Deployed to Firebase
firebase deploy
Live Website
